HTML elements

The HTML element is everything from the start tag to the end tag, for example <p> Content here... </p>.
Void elements are all elements that have no end tag for example <br>

Attributes

Ever element can have attributes. They change the way of how the element looks, works, interacts,etc...
Some examples are:
<src>(sets the source of an image),
<href>(sets the hyper reference for a link),
<alt>(puts an alt text for an image if it doesn't load),
<width>(sets the width of an element),
and many more...

Headings

Headings are very important. They are like a title for your documents. There are different sizes of heading.
The biggest is <h1> and the smallest is <h6>.

Paragraphs

A paragraph is defined using <p>. The browser automatically removes any spaces.
By using <br> you can make a line break and by using <hr> you can make a horizontal rule.
Using <pre> you can let text be the same as in your editor.

Comments

Comments are very useful. They can be used to add text on your editor without it showing on your website, This can be used to help others or your future self understand the code.
To use a comment, you just need to write: <!-- add comments here -->.

Links

Links in HTML are just simple hyperlinks. Use them to jump to other websites or documents.
The <a> defines an anchor, which can be used for links. Just add the href attribute to add a hyperlink.
Using the target attribute, you can choose how the documents get opened.
_self: This opens the link in the same window. This is also the default
_blank: This opens the link in a new window
_parent: This opens the link in a parent frame
_top: This opens the link in the full body of the window

Images

Images are very important to use. By using <img> you can add a image. The img tag is a void tag since it had no ending tag.
You need to also use 2 attributes: src and alt. (More on Attributes).
The alt attribute provides an alternative text,for when a user cannot see the images. .
If you have the image in another folder, then you have to use the correct path, for example <img src="/images/banana.png alt="This is a pic of a banana>
If the image is on another server, you would need to add the URL and path, for example <img src="https://www.w3schools.com/images/w3schools_green.jpg" alt="W3School>

Head

The <heade> element contains all the data about the document, like the title, language, scripts, etc.
The <title> element sets the name of the document and must be text-only. It is also required to use. The title is important for the Seach Engine Optimization (SEO).
The <meta> element is used to specify keywords, description, authot, etc.
